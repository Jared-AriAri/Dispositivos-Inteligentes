# Climate App

App Flutter creada para la práctica P2.1 de Dispositivos Inteligentes.

## Objetivo

Instalar y configurar el entorno Flutter, crear un primer proyecto llamado `climate_app`, ejecutarlo en un emulador Android y comprobar el funcionamiento de hot reload.

## Requisitos

- Flutter SDK
- Dart SDK
- Android Studio
- Android SDK
- Emulador Android o dispositivo físico

## Comandos utilizados

```bash
flutter doctor -v
flutter devices
flutter create climate_app
cd climate_app
flutter run -d emulator-5554
